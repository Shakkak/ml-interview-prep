# ml-interview-prep

ML study vault for Obsidian + Anki. 91 concept notes and 444 flashcards covering deep learning, statistics, and ML theory — from fundamentals to interview-level depth.

## Contents

- **`vault/`** — 91 Obsidian-compatible Markdown notes across 9 topics
  - Each note is split into **Fundamental / Intermediate / Advanced** sections
  - Topics: architectures, training, theory, math, probability, generative models, self-supervised learning, adaptation, data

## Topics

| Directory | Concepts |
|---|---|
| `architectures/` | Attention, CNNs, ViT, GNN, residual blocks, positional encodings, flash attention, ... |
| `training/` | Backprop, optimizers (SGD/Adam/RMSProp), normalization, regularization, loss functions, ... |
| `theory/` | Bias-variance, spectral bias, kernel methods, calibration, generalization bounds, ... |
| `math/` | Linear algebra, SVD, PCA, Fourier transform, matrix calculus, convexity, ... |
| `probability/` | Bayesian inference, MLE, entropy, distributions, hypothesis testing, sampling, ... |
| `generative/` | VAEs, GANs, diffusion models, normalizing flows, autoregressive models |
| `self-supervised/` | CLIP, BLIP, BERT, contrastive learning, knowledge distillation, ... |
| `adaptation/` | LoRA, RLHF, instruction tuning, prompt engineering, transfer learning |
| `data/` | Evaluation metrics, feature preprocessing, active learning, augmentation, ... |

## Anki Deck

Pre-built `.apkg` files are available in [Releases](../../releases) — download and import directly into Anki.

## Usage with Obsidian

Clone the repo and open the `vault/` folder as an Obsidian vault. Notes use `[[wikilinks]]` for cross-concept navigation.
